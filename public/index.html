FreeCodeCamp API Project 1: Timestamp Microservice

User stories:
    1) User can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016).
    2) If it does not contain a date or Unix timestamp, it returns null for those properties.
    3) If it does, it returns both the Unix timestamp and the natural language form of that date.

Example usage:

Copy & Paste the following URLs into your browser's address bar to see the result.
1. https://r282-timestamp-api.herokuapp.com/January%2017,%202017
2. https://r282-timestamp-api.herokuapp.com/1484611200


Example output:

The following displays the output from the examples above.

{ "unix": 1484611200, "natural": "January 17, 2017" }
